# Dokumentasi Tugas JavaScript

## Langkah Pengerjaan

1. **Buka file `index.js`**
   - Di dalam file terdapat beberapa potongan kode yang masih dalam bentuk *comment* (`//`).
   - Hapus tanda `//` pada satu potongan kode, lalu simpan.

2. **Jalankan di browser**
   - Buka file `index.html` menggunakan browser.
   - Tekan **F12** atau klik kanan → **Inspect** → pilih tab **Console**.
   - Perhatikan hasil output dari kode yang dijalankan.

3. **Dokumentasikan hasil**
   - Untuk setiap potongan kode:
     - Ambil **screenshot kode** di `index.js`.
     - Ambil **screenshot hasil output** di Console.
     - Masukkan keduanya ke dokumen Word.
     - Tambahkan **penjelasan singkat**, meliputi:
       - Fungsi dari kode tersebut.
       - Alasan mengapa output sesuai/berbeda dengan yang diharapkan.

4. **Kumpulkan hasil**
   - Setelah semua potongan kode diuji, simpan dokumen Word sebagai **PDF**.
   - File PDF inilah yang dikumpulkan sebagai hasil akhir.

---

## Saran Format Dokumentasi

Gunakan format tabel di Word agar lebih rapi:

| Potongan Kode | Output di Console | Penjelasan |
|---------------|------------------|------------|
| Screenshot kode | Screenshot output | Penjelasan singkat |

---

## Catatan
- Jalankan kode **satu per satu** (jangan sekaligus).  
- Pastikan setiap hasil sudah terdokumentasi sebelum lanjut ke potongan kode berikutnya.
